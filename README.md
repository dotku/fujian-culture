# 福建文化网 (演示)

福建文化网是一个致力于展示和传播福建省丰富文化遗产的在线平台。本网站由福建省文化厅主办，福建省图书馆承办与制作。

## 主要内容

### 文化动态

及时发布福建省各地文化活动、展览和重要新闻，让您随时了解福建文化发展的最新动态。

### 文化展览

展示福建省各类文化展览活动，包括传统艺术展、非物质文化遗产展等。

### 讲坛荟萃

提供高质量的文化讲座和学术报告，邀请各领域专家学者分享见解。

### 数字图书馆

提供丰富的数字资源，包括：

- 电子期刊（维普中文科技期刊、万方数字化期刊）
- 地方文献数据库（方志、闽人著述）
- 家谱文献（联合家谱、馆藏家谱）
